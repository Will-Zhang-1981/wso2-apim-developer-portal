Kom igång

På denna sida har vi sammanställt information om vad som krävs för att snabbt komma igång med att använda utvecklarportalen. Skapa konto, applikationer och att prenumerera på våra API:er beskrivs i enkla steg.

Skapa konto

Skapa konto för utvecklarportalen når du på två sätt: via knapp Skapa konto på startsidan eller menyalternativet Logga in -> Skapa konto.

I formuläret som visas fyller du i efterfrågade uppgifter och väljer Skapa.

Ett epostmeddelande kommer nu att skickas till den epostadress du angav i formuläret. Klicka på den bifogade länken för att visa aktiveringssidan.

Fyll i användarnamn och lösenord och skriv in det aktuella värdet för captcha och välj "SKAPA/VERIFIERA". Dina uppgifter verifieras och givet att du fyllt i korrekt information aktiveras ditt konto och du har nu tillgång till alla tjänster på utvecklarportalen.

Skapa applikation

Skapa applikation når du via alternativet Mina Applikationer i profilmenyn. Du dirigeras då till sidan med en förteckning över redan skapade applikationer.

Fyll i nödvändiga uppgifter och välj Skapa. Din nya applikation finns nu tillgänglig i listan överst på sidan.

Skapa prenumeration

Observera att du måste vara inloggad och ha en applikation för att kunna skapa en prenumeration. Utvecklarportalen erbjuder två vägar att gå för att skapa prenumeration.

  1. Via detaljsidan för ett API
  2. Via sidan Mina prenumerationer
