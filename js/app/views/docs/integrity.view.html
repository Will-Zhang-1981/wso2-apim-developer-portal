Konto - Utvecklarportalen

Att skapa ett konto på Utvecklarportalen är nödvändigt för att kunna ta del av tjänsterna som erbjuds. Om du väljer att lämna dina personuppgifter är Västtrafik ansvarig för dem. Informationen kommer att behandlas konfidentiellt och enligt Personuppgiftslagen. Kontorelated information som vi sparar är förnamn, efternamn, epostadress och lösenord vilket är nödvändigt för att Du ska komma åt våra tjänster.

Till vad används kontouppgifter?

Lämnad information används enbart till att ge dig tillgång till samtliga tjänster på Utvecklarportalen.

Kan mina uppgifter spridas?

Dina uppgifter används bara för att ge dig tillgång till dit konto här på Utvecklarportalen.
