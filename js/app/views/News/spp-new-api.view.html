Smart pendelparkering är ett API vi har haft sedan 2014, men inte lanserat officiellt som ett öppet API. Vi tar nu tillfället i akt att erbjuda detta API till alla som vill använda det! Genom att använda detta API får du tillgång till information om våra parkeringsplatser för pendlare. De funktioner du hittar i detta API omfattar:

  • Parkeringsområden. Möjlighet att söka efter parkeringsområden baserat på kommun, hållplatsområde eller koordinat. I resultatet kan man se de olika parkeringsplatser och hållplatser som hör till området.
  • Lediga parkeringsplatser. Möjlighet att få reda på antal lediga platser i realtid för ett parkeringsområde eller enskild parkeringsplats.
  • Bilder från parkeringsplatser. För de parkeringsplatser där det finns kamera finns möjlighet att få en bild i realtid (se nedan). Observera att bilderna är ”maskade” p.g.a. integritetsskäl.
  • Statistik. Möjlighet att fråga efter tillgänglighet för ett visst parkeringsområde eller enskild parkeringsplats under en viss tidsperiod.
